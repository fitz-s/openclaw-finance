# Reports (v0)

All generated briefings live here.

## Naming
`YYYY-MM-DD_<phase>_<market>.md`
- phase: pre | open | post | event
- market: us | global | crypto

## Format (v0)
1. Context
2. Watchlist Moves
3. Events / Filings
4. Risks / Unknowns (To Verify)
5. Next Checks (optional)

## Note
Reports are generated by renderer cron jobs triggered on-demand by gate_evaluator.py.
